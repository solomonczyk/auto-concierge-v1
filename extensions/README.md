# Extensions

Каталог для клиентских расширений и кастомных интеграций.

Правило архитектуры:

- Core — универсальная логика платформы
- Extensions — клиентские интеграции, нестандартные workflow, внешние системы

Core не должен зависеть от extensions.
Extensions могут использовать API и сервисы core.
