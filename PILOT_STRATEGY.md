# Цель пилота
Подключить 3 автосервиса в РФ и получить первые платные внедрения.

# Формат
Ручное подключение
Персональная настройка
Фиксированная цена внедрения

# Тип клиента
Небольшие автосервисы и СТО
3–10 механиков
Без собственной CRM

# Что решает система
— онлайн запись клиентов  
— автоматическая запись через Telegram  
— календарь загрузки сервиса  
— напоминания клиентам  

# Формат пилота
Подключение вручную
Настройка под сервис
Поддержка напрямую

# Цель пилота
Проверить:
— готовы ли сервисы платить
— какие функции реально используются
— какие доработки нужны

# Критерии отбора сервисов

1. Автосервис принимает клиентов по записи
2. У сервиса есть Telegram или WhatsApp
3. Администратор отвечает на сообщения
4. Сервис не использует сложную CRM
5. В сервисе 3–10 механиков

# Где искать первые сервисы

1. Яндекс.Карты
Поиск: "автосервис", "СТО", "замена масла"

2. 2ГИС
Фильтр по рейтингу 4.0–4.8

3. Avito
Раздел: услуги → автосервис

4. Telegram
Локальные городские каналы и чаты автомобилистов

5. Сайты автосервисов
Если указан WhatsApp или Telegram — это хороший кандидат

# Первый контакт с сервисом

Здравствуйте.

Я занимаюсь внедрением системы онлайн-записи для автосервисов.
Она позволяет клиентам записываться через Telegram и сайт, а администратор видит всё в одном календаре.

Сейчас подключаю несколько сервисов в пилотный запуск.

Могу бесплатно показать, как это работает и как это может выглядеть для вашего сервиса.

Если будет полезно — внедрим.

Показ занимает 10–15 минут.
